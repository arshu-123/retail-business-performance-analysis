# retail-business-performance-analysis

# Retail Business Performance and Profitability Analysis

## Objective

Analyze Walmart retail sales data to evaluate sales performance, profitability, and customer trends using SQL, Python, and Power BI.

## Dataset

* Walmart Sales Dataset

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn
* SQL
* Power BI

## Features

* Data exploration and preprocessing
* SQL-based sales analysis
* Product line and city-wise performance evaluation
* Monthly sales trend analysis
* Customer type profitability analysis
* Interactive Power BI dashboard

## Project Files

* `Retail_Analysis.ipynb`
* `sales_analysis.sql`
* `dashboard.pbix`
* `dashboard.png`
* `Retail_Business_Performance_Report.pdf`
* `Walmart sales dataset cleaned (new).csv`

## Key Insights

* Naypyitaw recorded the highest sales.
* Evening sales outperformed other time periods.
* Food and Beverages and Fashion Accessories ranked among the top product lines.
* Member customers generated higher gross income.

## Outcome

The project demonstrates end-to-end retail data analysis using SQL, Python, and Power BI to support informed business decisions.
